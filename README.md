# Usage
<!-- usage -->
```sh-session
$ npm install -g react-native-assistant
$ rna COMMAND
running command...
$ rna (-v|--version|version)
react-native-assistant/0.0.1 darwin-x64 node-v10.16.3
$ rna --help [COMMAND]
USAGE
  $ rna COMMAND
...
```
